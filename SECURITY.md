# Security

Please do not report security issues in public GitHub issues.

After this repository is public, use GitHub Security Advisories for this
repository. Include:

- affected version or commit,
- operating system,
- reproduction steps,
- expected impact,
- any logs or sample files needed to reproduce safely.

PlanMaxx is local-first, but review content and side-question prompts may include
sensitive plan details. Treat saved review autosave files as local private data.
